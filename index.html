L's Cheat Sheet

HTML Tables
Tag Description Notes
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

HTML Tags
Tag Description
<figure> Use to mark up a photo in document, and a <figcaption> element to define a caption for the photo
<figcaption> Use to define a caption for the photo
<article> Specifies independent, self-contained content, finds use in forum, blog or news post sections
<blockquote> Specifies a section that is quoted from another source
<caption> Defines a table caption, must be inserted immediately after the table tag

CSS Tips
Topic Description
Display Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted.
Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted.
Inline-block elements can appear next to each other, and can have their width and height manually adjusted.
Clear The clear property specifies how elements should behave when they bump into each other on the page.
Vertical Margins Unlike horizontal margins, vertical margins do not add. Instead, the larger of the two vertical margins sets the distance between adjacent elements.
